# Portfolio Website - Riya Chowdhury

A dark-themed, interactive portfolio website built as a web development assignment.

## 🚀 Features

- **Interactive Background**: A dynamic dot-matrix canvas that reacts to mouse movements with repulsion and click-scatter effects.
- **Responsive Design**: Fully responsive layout that adapts to desktops, tablets, and mobile phones, including a mobile hamburger menu.
- **Modern UI/UX**: Uses a sleek dark theme with a light violet accent, glassmorphism (semi-transparent boxes), and elegant cursive typography.
- **Downloadable Resume**: A dedicated button to easily download the resume in PDF format.
- **Sections Included**: Home, About Me, Skills, Projects, Education, and Contact.

## 🛠️ Technologies Used

- **HTML5**: For structuring the content.
- **CSS3**: For styling, responsive design (Flexbox & CSS Grid), and animations (hover effects, glowing borders).
- **JavaScript (Vanilla)**: For the interactive canvas background and smooth scrolling navigation.

## 💻 How to Run Locally

To view this project locally, simply download or clone the repository and open the `index.html` file in any modern web browser. No server or compilation steps are required.

```bash
git clone https://github.com/RiyaGithub123/portfolio_website.git
```
Then, double-click on `index.html`.
